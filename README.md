# Customer Churn Analysis

## Project Overview
Predicted customer churn using Telco dataset (7,043 customers).

## Key Findings
- 26% of customers churned
- Model Accuracy: 80.70%
- Customers with high monthly charges are more likely to churn

## Tools Used
- Python, Pandas, Matplotlib, Scikit-learn

## Files
- churn.py - Main analysis code
- churn_chart.png - Visualization
